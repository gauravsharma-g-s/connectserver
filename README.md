The backend of **Connect** is a RESTful API and real-time server built using the **MERN stack**.  
It handles authentication, user management, posts, friends, real-time chat, and media uploads.

---

## Features

- User Authentication (Register & Login)
- Password hashing using **bcrypt**
- JWT-based authorization
- Create, like/unlike, and comment on posts
- Add and remove friends
- Real-time chat using **Socket.io**
- Image uploads using **Cloudinary**
- Secure API with **Helmet**
- Request logging with **Morgan**
- Email support using **Nodemailer**
- MongoDB integration with **Mongoose**
